# Quality Gates — ThD Research System

This file defines when a note is allowed to move upward in the system.

Notes move through these stages:

captured → processed → linked → tested → vetted → chapter-ready

---

## Gate 1 — Captured
Purpose: raw intake.

A note is captured if:
- it exists
- content is recorded (even rough)

✅ Allowed:
- raw excerpts
- initial bullets

❌ Not allowed:
- using this note in arguments or synthesis

---

## Gate 2 — Processed
Purpose: clarity.

A note is processed if:
- it has a clear main idea or observation
- it is no longer raw copy-paste
- it fits one note type (concept, passage, patristic, etc.)

✅ You can now understand the note standalone.

---

## Gate 3 — Linked
Purpose: integration.

A note is linked if:
- it connects to ≥2 other notes
- links are meaningful (not decorative)

For patristic notes specifically:
- ≥1 Scripture link
- ≥1 Concept link

✅ You can now navigate to/from this note.

---

## Gate 4 — Tested
Purpose: intellectual rigor.

A note is tested if:
- it includes at least one opposing or alternative view
- that view is stated fairly and recognizably
- at least one weakness or tension is identified

✅ You have pressure-tested the idea.

---

## Gate 5 — Vetted
Purpose: reliability.

A note is vetted if:
- it has traceable sources (text, passage, or author)
- claims are not speculative without labeling
- argument structure is visible (not vague statements)

## Argument-Specific Gate

An argument note is not vetted unless it includes:
- claim
- qualifier
- grounds
- warrant
- backing
- opposing view
- rebuttal
- weakness / tension
- related source notes

An argument note is not chapter-ready unless:
- the warrant is explicit
- the opposing view is stated fairly
- the rebuttal answers the strongest version of the objection
- the claim is linked to at least one synthesis or chapter note

✅ You can trust this note for serious work.

---

## Gate 6 — Chapter-Ready
Purpose: writing usability.

A note is chapter-ready if:
- it has:
  - clear claim
  - grounded evidence
  - opposing view
  - defined implications
- it can support a section of a dissertation without embarrassment

✅ You can now use this in chapter construction.

---

## Core Rule

Do NOT promote notes early.

If a note skips gates:
→ arguments become weak  
→ synthesis becomes biased  
→ chapters become fragile  

The strength of the dissertation depends on respecting these gates.
