# Methodology

## Overall Method
This project uses a source-grounded patristic-theological method that moves from:
primary source reading → biblical grounding → argument analysis → theological synthesis → chapter construction.

## Exegetical Method
For biblical passages:
- define text boundaries
- observe structure
- identify QUD
- distinguish explicit and implicit claims
- avoid using passages as prooftexts without context

## Patristic Method
For patristic texts:
- identify historical context
- reconstruct the opponent fairly
- trace the author’s argument structure
- distinguish Scripture, worship, and theological inference
- avoid treating patristic language as mere philosophy or mere devotion

## Argument Method
Arguments must include:
- claim
- qualifier
- grounds
- warrant
- backing
- opposing view
- rebuttal
- implication

## Synthesis Method
Syntheses must:
- gather vetted notes
- separate biblical and patristic evidence
- preserve tensions
- state confidence level
- identify what would strengthen the argument

## Quality Control
Use [[Quality_Gates]] before promoting notes into chapters.

## Limits
This project does not assume that patristic worship automatically proves doctrine. It tests how worship functions within a broader scriptural and ecclesial argument.
