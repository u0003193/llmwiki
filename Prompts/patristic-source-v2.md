## PROCEDURE

### 1. Identify the Text Precisely
- author
- work
- section / homily / chapter
- edition / translation used

### 2. Establish Context
- historical controversy
- purpose of the text
- genre of the writing

### 3. Extract Core Ideas
Only those actually grounded in the text.

### 4. Relate to Scripture
Classify relationship as:
- explicit citation
- allusion
- theological inference
- conceptual appropriation

### 5. Note Key Terms
Greek / Latin terms if present and useful.

### 6. Record Reliability Limits
- translation limitations
- uncertain attribution
- selective reading risk
- need for better edition
