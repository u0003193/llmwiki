## EXTRACTION CATEGORIES

Extract only what is well grounded in the source:

### A. Bibliographic / Source Data
- author
- title
- date
- edition / translator if present
- page numbers or sections if available
- type of source: primary / secondary

### B. Claims
- central thesis
- major supporting claims
- tensions or unresolved issues

### C. Concepts
- technical terms
- recurring theological categories
- conceptual contrasts
- disputed definitions

### D. Persons
- ancient authors
- modern scholars
- biblical figures only if conceptually central

### E. Biblical References
- explicit citations
- strong allusions only if clearly justified

### F. Patristic References
- named authors
- cited works if available

### G. Questions
- explicit questions
- implicit questions required by the argument

### H. Opposing or Alternative Views
- explicitly stated opposing views in the source
- implicit disagreements or tensions

---

## NOTE CREATION RULES

### Before creating a note
Check whether an equivalent note already exists.
If yes:
- update it
- do not duplicate it
- preserve useful prior content

### If creating a new note
Keep it atomic and reusable.
One note = one concept / question / passage / person.

---

## REQUIRED DISCIPLINE

For each extracted claim distinguish:
- what the source says
- what the source implies
- what remains uncertain

Do not:
- invent citations
- force harmonization
- turn extraction into constructive theology
- collapse multiple concepts into one note
