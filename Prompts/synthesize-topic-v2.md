Purpose

Build a constructive synthesis from already-vetted evidence and argument notes. This prompt is stricter than the original because it explicitly refuses premature synthesis and requires naming weak evidence areas.

Final Prompt
## GOAL
Create a grounded theological synthesis from vetted notes.

## CONTEXT
This synthesis must remain traceable to evidence.
It must preserve tensions and distinguish confidence from speculation.

## SOURCE
Use:
- relevant notes in Literature/, Concepts/, Passages/, Persons/, Patristics/, Questions/, Arguments/, MOCs/
- Quality_Gates.md
- COPILOT-INSTRUCTIONS-v2.md

## EXPECTATIONS
Return:
1. synthesis note path
2. populated note sections
3. weak evidence areas
4. unresolved tensions

## OUTPUT TARGETS

Create or update:

- Syntheses/

May update:

- Arguments/
- Concepts/

Never create:

- Literature/
- Sources/
- Passages/
- Chapters/

Reason:
This prompt integrates evidence but does not create chapter material.

---

## PROCEDURE

### 1. Gather the Evidence Base
List the notes used.

### 2. Separate Layers
Distinguish:
- biblical evidence
- patristic evidence
- later theological development if present

### 3. Identify Competing Views
Before constructing synthesis:

- clearly state the main competing or alternative interpretations
- ensure each view is recognizable and fairly represented
- identify the strongest argument for each competing view

### 4. Identify Agreements
What converges?

### 5. Identify Tensions
What remains contested?

### 6. Build Constructive Synthesis
State the best current integrative account.

### 7. Rate Confidence
low | medium | high

### 8. Identify What Would Strengthen the Synthesis
What additional passages, fathers, or scholars are still needed?

---

## RULES

- Do not merely summarize one note after another
- Do not erase disagreement
- Do not speak more confidently than the evidence allows
- Name where the evidence base is thin
