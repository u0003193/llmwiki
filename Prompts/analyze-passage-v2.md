A missing but necessary prompt for Version 2. Helps the student move from pericope boundaries to structure, text issues, QUD, and thesis before theological synthesis. This reflects the stronger exegetical workflow recommended in Exegetical Paper Guide.

Final Prompt
## GOAL
Perform a disciplined exegetical analysis of one biblical passage.

## CONTEXT
The purpose is to understand how the passage works before using it in theological argument.

## SOURCE
Use:
- the passage itself
- surrounding literary context
- existing note network
- COPILOT-INSTRUCTIONS-v2.md

## EXPECTATIONS
Return:
1. text boundaries
2. structural outline
3. key observations
4. QUD analysis
5. provisional thesis
6. open questions

## OUTPUT TARGETS

Create or update:

- Passages/
- Questions/

May update:

- Concepts/

Never create:

- Arguments/
- Syntheses/
- Chapters/

Reason:
Passage analysis should remain exegetical before theological construction.

---

## PROCEDURE

### 1. Define the Text Unit
State where the passage begins and ends.
Defend the boundaries briefly.

### 2. Compare Major Translations
Identify major translation differences that might affect meaning.

### 3. Note Textual Issues
Flag important variants if they materially affect interpretation.

### 4. Observe Structure
Outline the passage.
Note conjunctions, contrast, repetition, inclusio, progression, grounds, and purpose.

### 5. Explain Argument / Narrative Flow
Show how the parts relate.

### 6. Formulate QUD
Identify:
- explicit question
- implicit question
- answer
- grounds
- implication
- confidence level

### 7. Draft Provisional Section Thesis
One sentence only.

### 8. Identify Open Questions
What remains uncertain?
What additional evidence is needed?

---

## RULES

- Do not move to broad theology too quickly
- Do not pretend all translation issues matter equally
- Keep observation separate from interpretation
- If original-language work is absent, say so
